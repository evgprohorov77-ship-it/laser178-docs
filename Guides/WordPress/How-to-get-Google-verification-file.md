# Как скачать файл подтверждения Google Search Console

## Пошаговая инструкция

### Шаг 1. Откройте Google Search Console

1. Перейдите по адресу: [https://search.google.com/search-console](https://search.google.com/search-console)
2. Войдите в свой Google-аккаунт ( Gmail).

### Шаг 2. Начните добавление сайта

1. На главной странице нажмите кнопку **«Добавить ресурс»** или **«Начать»**.
2. Откроется окно с двумя вкладками:
   - **Домен** — для подтверждения всего домена
   - **URL-префикс** — для подтверждения только одного URL

### Шаг 3. Выберите «URL-префикс"

1. Перейдите на вкладку **«URL-префикс»**.
2. В поле введите: `https://laser178.ru/`
3. Нажмите кнопку **«Продолжить»**.

### Шаг 4. Выберите способ подтверждения «HTML-файл"

1. Откроется страница **«Подтвердите права собственности"**.
2. Найдите блок **«Рекомендуемый метод»** или список способов.
3. Выберите **«HTML-файл»**.
4. Вы увидите строку с именем файла, например:
   ```
   google1234567890123456.html
   ```
5. Нажмите кнопку **«Скачать»** (или стрелку вниз рядом с именем файла).
6. Файл сохранится на ваш компьютер.

### Шаг 5. Откройте файл и скопируйте содержимое

1. Откройте скачанный файл в любом текстовом редакторе (Блокнот, TextEdit).
2. Там будет примерно такой текст:
   ```
   google-site-verification: google1234567890123456
   ```
3. Скопируйте **имя файла** и его **содержимое**.

### Шаг 6. Отправьте ассистенту

1. Пришлите в чат имя файла, например:
   ```
   google1234567890123456.html
   ```
2. Пришлите содержимое файла, например:
   ```
   google-site-verification: google1234567890123456
   ```

Ассистент загрузит файл в корень сайта `laser178.ru`, после чего можно будет нажать **«Подтвердить»** в Google Search Console.

---

## Если вы не видите способ «HTML-файл"

Google иногда скрывает альтернативные способы. Чтобы найти **«HTML-файл»**:

1. На странице подтверждения прокрутите вниз.
2. Найдите ссылку **«Другие способы проверки»** или **«Другие методы подтверждения»**.
3. Нажмите на неё.
4. В открывшемся списке выберите **«HTML-файл»**.

---

## Что делать после загрузки файла

1. Вернитесь в Google Search Console.
2. Нажмите кнопку **«Подтвердить»**.
3. Если всё сделано правильно, Google покажет сообщение: **«Права подтверждены»**.
4. После этого можно добавить sitemap:
   - В левом меню выберите **«Sitemap»**.
   - Введите: `sitemap_index.xml`
   - Нажмите **«Отправить»**.

---

## Альтернативный способ: DNS-запись

Если файл не подходит, можно подтвердить через **DNS-запись** (вкладка **«Домен»**):

1. Выберите вкладку **«Домен»**.
2. Введите: `laser178.ru`
3. Нажмите **«Продолжить»**.
4. Google покажет TXT-запись.
5. Скопируйте её и пришлите ассистенту — он даст точную инструкцию для Beget.
